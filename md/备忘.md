### Win 自启动文件夹

- 在 Win+r 里运行：
```terminal
    >|shell:startup
```

### 格式化 css

- 确保你安装了合适的 CSS 格式化扩展，比如 Prettier 或 CSS Formatter。<br>
安装后，你可以在命令面板（Ctrl + Shift + P 或 Cmd + Shift + P）中运行 Format Document 来格式化代码。
```terminal
    >|Format Document
```
